## Loader

### Usage

```tsx
// Basic usage with default variant (spinner)
<Loader />

// Different variants
<Loader variant="spinner" />
<Loader variant="processing" />
<Loader variant="installing" />
<Loader variant="dots" />

// Different sizes
<Loader size="sm" />
<Loader size="md" />
<Loader size="lg" />

// Custom labels
<Loader label="Saving data..." />

// Label position
<Loader label="Please wait" labelPosition="left" />

// Custom styling
<Loader className="text-primary" />
```
